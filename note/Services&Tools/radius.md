## Protocol
### what is  radius
- Radius: Remote Authentication Dial In User Service
- It is a protocol for AAA(Authentication,authorization,accounting)
- worked at  c/s mode
- [rfc2865](https://tools.ietf.org/html/rfc2865)

### Details of radius
- base on udp
- radius packets:
   - Access-Request
   - Access-Accept
   - Access-Reject
   - Access-Challenge
